# lemoncloud-js

npm 모듈 생성하고, npmjs에 모듈 등록하는 방법.

```bash
# create folder
$ mkdir lemoncloud-js
$ cd lemoncloud-js

# initialize package.json
$ npm init

# edit README
$ echo "# lemoncloud-js" >> README.md

# npm login (register if not)
$ npm login 

# publish
$ npm publish
```

참고: https://eladnava.com/publishing-your-first-package-to-npm/
